# Grasshopper Live Bridge 🚀

A three-tier system for live coding and AI-driven design in Grasshopper.

## Components

1. **Grasshopper Component** (`/grasshopper-component`) - WebSocket server running in GH
2. **VSCode Extension** (`/vscode-extension`) - Live Python editing interface  
3. **MCP Server** (`/mcp-server`) - AI orchestration layer [Coming Soon]

## Quick Start

1. Build and install the Grasshopper component
2. Install the VSCode extension
3. Start coding!

See individual component READMEs for detailed instructions.
